# Sitios Eventos y Turismo

Proyecto React para la agencia de viajes "Sitios Eventos y Turismo".

## Instalación

1. Asegúrate de tener Node.js instalado (versión 16 o superior). Si no lo tienes, descárgalo desde [nodejs.org](https://nodejs.org/).

2. Instala las dependencias:
   ```
   npm install
   ```

## Ejecución

Para iniciar el servidor de desarrollo:
```
npm start
```

Esto abrirá la aplicación en http://localhost:3000.

## Construcción

Para construir la aplicación para producción:
```
npm run build
```

## Notas

- Utiliza Tailwind CSS para estilos (incluido vía CDN en index.html).
- React 18 está configurado.
